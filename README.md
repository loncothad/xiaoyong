# xiaoyong

效用 (Xiào yòng) means "utility" in Chinese.

A collection of Rust utility crates that I personally needed but couldn't find
implemented anywhere. They are located at the [`crates/`](crates/) subfolder.
Check the respective READMEs of the crates for more info about each one.
